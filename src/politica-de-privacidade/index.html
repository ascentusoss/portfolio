Política de Privacidade

Esta Política de Privacidade está em conformidade com a Lei Geral de Proteção de Dados (LGPD - Lei nº 13.709/2018) e descreve como tratamos seus dados pessoais.

1. Visão Geral

Este é um portfólio pessoal open source. Eu (Ítalo C Lopes) respeito sua privacidade e busco coletar o mínimo possível de dados. Esta página explica como os dados são tratados quando você acessa este site.

2. Dados que Coletamos

2.1. Dados de Navegação (Logs)

Como qualquer site, dados técnicos podem ser registrados automaticamente pela plataforma de hospedagem:

  • Endereço IP
  • Tipo de navegador e sistema operacional
  • Data e hora de acesso
  • Páginas acessadas

Base legal: Legítimo interesse (Art. 7º, IX da LGPD) para segurança e operação do site.

2.2. Cookies e Tecnologias Similares

Este site utiliza:

  • Vercel Analytics: Para métricas de desempenho
  • Vercel Speed Insights: Para otimização de performance

Você pode desabilitar cookies nas configurações do seu navegador.

2.3. Dados de Contato

Se você enviar email ou mensagem via WhatsApp, receberemos:

  • Seu nome ou identificação
  • Endereço de email ou número de telefone
  • Conteúdo da mensagem

Esses dados são usados apenas para responder sua solicitação.

3. Como Usamos os Dados

  • Responder mensagens e solicitações de contato
  • Manter o site seguro e operacional
  • Melhorar a qualidade e conteúdo do site
  • Análise geral de desempenho

4. Compartilhamento de Dados

Não vendemos seus dados. Dados podem ser compartilhados apenas com:

4.1. Vercel (Hospedagem)

O site é hospedado na Vercel. Dados de logs e analytics podem ser processados conforme a Política de Privacidade.

5. Armazenamento e Segurança

5.1. Localização dos Dados

Dados podem ser armazenados em servidores nos Estados Unidos (Vercel), com certificações de segurança internacionais.

5.2. Medidas de Segurança

  • HTTPS/TLS para criptografia de dados em trânsito
  • Logs acessados apenas para operação e segurança

5.3. Retenção de Dados

  • Logs de acesso: Conforme política da Vercel
  • Mensagens de contato: Mantidas enquanto necessário
  • Analytics: Conforme políticas do Vercel Analytics

6. Seus Direitos (LGPD)

De acordo com a LGPD, você tem direito a solicitar:

  • Confirmação se seus dados são processados
  • Cópia dos dados que temos sobre você
  • Correção de dados incompletos ou desatualizados
  • Exclusão de dados desnecessários
  • Portabilidade de dados em formato estruturado

Para exercer esses direitos, entre em contato através dos canais no rodapé.

7. Links Externos

Este site pode conter links para GitHub e outros serviços externos. Não somos responsáveis pelas políticas de privacidade desses sites.

8. Menores de Idade

Este site não é direcionado a menores de 13 anos. Se você é responsável por uma criança que forneceu dados, entre em contato.

9. Transferência Internacional

Dados podem ser processados fora do Brasil (EUA) em conformidade com os requisitos da LGPD através de provedores com certificações de segurança reconhecidas.

10. Alterações a Esta Política

Esta política pode ser atualizada periodicamente. A data no topo indica a versão mais recente.

11. Contato

Para questões sobre privacidade ou para exercer seus direitos, entre em contato através dos canais disponíveis no rodapé do site (GitHub ou WhatsApp).

Compromisso: Estou comprometido em proteger sua privacidade e garantir a transparência no tratamento de dados pessoais, em total conformidade com a LGPD.
